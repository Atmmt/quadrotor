# 四旋翼无人机 - STC / CYT4BB7 电调 + FreeRTOS 飞控

## 项目简介

本项目是一个**自制四旋翼无人机**的完整软硬件方案，包含：

- **飞控主控**：基于 FreeRTOS 的实时操作系统，实现姿态解算、传感器融合、PID 控制、通信协议解析等任务。
- **电调设计**：
  - **STC 电调**：基于 STC 系列单片机的低成本电调，支持 I²C / PWM 控制信号。
  - **CYT4BB7 电调**：基于 Infineon (Cypress) CYT4BB7 芯片的高性能电调，支持更高的更新率和电流输出。
- 支持多旋翼常见外设（MPU9250 或 ICM42688 等 IMU、GPS、气压计、无线数传等）。

> **CYT4BB7** 是英飞凌 PSoC™ 4 系列的一款 Arm® Cortex®-M0+ 微控制器，主频可达 48 MHz，集成可编程模拟/数字模块，非常适合高性能电调控制。
