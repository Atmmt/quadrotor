# 四旋翼无人机 - CYT4BB7 飞控

## 项目简介

本项目是一个**完全自制的四旋翼无人机系统**，包括：

- **飞控**：基于 **Infineon CYT4BB7** 双核芯片（Cortex‑M7 @ 300MHz + Cortex‑M0+ @ 100MHz），运行 **FreeRTOS** 实时操作系统，负责姿态解算、传感器融合、PID 控制、遥控器解码，并通过 **Dshot600** 或 **PWM** 协议向电调发送指令。
- **电调**：四路电调均采用 **STC AI8051U**（增强型 8051 内核，集成硬件比较器），使用**六步换向法 + 反电动势零点检测**实现无传感器无刷电机控制，支持 Dshot600 数字协议或传统 PWM 模拟信号。
- **通信协议**：飞控通过 定时器PWM波产生 Dshot600 帧（或 PWM 脉宽），STC 电调解码后驱动电机。
